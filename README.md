# Relay Logistics Load Calculator V3.8

## Updates

- Adds a separate app-wide Pay Percentage Settings section.
- Company Driver % and Owner Operator % can be updated in one place.
- Updated percentages apply to both:
  - Load Quote
  - Completed Loads / Driver Profit
- Per-load calculation still saves the percentages used at the time of entry.
- Owner-operator toll chargeback and multi-stop pickup/delivery fields remain.

## Supabase

No new database columns are required if you already ran V3.7 SQL.
